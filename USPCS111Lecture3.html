Variable Names

  • Variables should have a name that describes what they contain.
  • age is good.
  • age_of_character is better in many contexts.
  • variable names must start with a letter.
  • variable names must not contain a space or any of $%*&@.
  • In most companies variable names must be in English.

Using Variables

#include <string>
#include <iostring>

int main() {
  uint age(21);
  std::string name ("Hiro Protagonist");
  
  std::cout << name << " is " << age << " years old" << std::endl;
}
