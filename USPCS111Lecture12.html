User Defined Nouns

  • We can use the idea of a class to define our own types.
  • Give our type a name (like, std::string, but not that because that name is already taken).
    • if we're trying to read records from a CSV file, then we could call our type CSVRecord?

An example

class CSVRecord {
public:
  std::string name;
  std::string value;
};

Dot notation

  • Note that our type is made up of other things.
  • We can use the . to access any of the public bits.
    • these public bits tend to be called fields.
    • So my_record.name accesses the name field of a thing called my_record
    • if the field doesn't exist, you'll get a compile-time error :)
