Usage

Demo time!

Dot notation

  • Note that our type is made up of other things.
  • We can use the . to access any of the public bits.
    • these public bits tend to be called fields.
    • So my_record.name accesses the name field of a thing called my_record
    • if the field doesn't exist, you'll get a compile-time error :)
