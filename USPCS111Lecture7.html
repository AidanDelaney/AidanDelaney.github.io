std::vector and functions

We may:

  • pass a std::vector to a function.
  • return a std::vector from a function.

std::vector parameters

void add_values(std::vector<int> values) {
    values.push_back(7);
}

int main() {
  std::vector<int> values;
  add_values(values);
  return 0;
}

return std::vector

std::vector<int> do_stuff() {
    std::vector<int> my_vec;
    my_vec.push_back(42);
}

int main() {
  std::vector my_vec = do_stuff();
  return 0;
}

Loops

Loops and lists

An example of a C++11 for loop is the following:

std::vector<char> letters = {'a','b','c'};

for(auto c: letters) {
  std::cout << c << std::endl;
}

auto

The auto keyword deduces the type!

  • introduced in C++11
  • significantly helps readability.
  • Where the compiler can know the type (eg: it was previously declared) then auto can work it out.

for

This form of a for loop is probably better named the for each loop.

  • for each element in letters
    • do something

Also introduced in C++11 (aside on making C++11 work in Dev-C++)

Events

  • Suppose we don't have a bound to iterate over
    • i.e. from the start to the end of a vector
  • How do we wait until some event happens?
  • We formulate the event as a condition.

Conditions

We have seen these before:

x == y  // x equals y
x != y  // x is not equal to y
x > y   // x is greater than y
x < y   // x is less than y
x >= y  // x is greater than or equal to y
x <= y  // x is less than or equal to y

while

while (x==y) {
  // do something
}
